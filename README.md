# wikiSearch
#### an elegant search engine for wikipedia

![wiki Search](assets/wikiSearch.png)

To view simply run these commands in your terminal: <br>
1. ```cd desktop```<br>
2. ```git clone https://github.com/danielrussellLA/wikiSearch.git```<br>
3. ```cd wikiSearch``` <br> 
4. ```npm i```<br>
5. ```open index.html```
